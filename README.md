# Exploring Conflict-free Replicated Data Types (CRDT) with Hybrid Logical Clocks (HLC)

### Instructions

1. Clone this project.

2. Run `npm ci`

3. Start the server:

```bash
node ./server.js
```

4. Open `index.html` in a web browser.
